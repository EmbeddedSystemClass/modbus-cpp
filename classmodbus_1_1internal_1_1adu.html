ADU structure for TCP:

  • Header (7 bytes) — Transaction (2 bytes) — Protocol (2 bytes) — Length (2 bytes) [ unit identifier + PDU ] — Unit (1 byte)
  • PDU — Function (1 byte) — Rest of data... (N byte)
